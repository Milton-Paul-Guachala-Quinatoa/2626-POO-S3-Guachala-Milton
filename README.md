## Descripción

Este repositorio contiene dos programas en Python que resuelven
el mismo problema (gestión de mascotas) usando dos enfoques distintos.

- **programacion_tradicional/**: Usa variables y funciones, sin clases.
- **programacion_poo/**: Usa la clase `Mascota` con atributos y métodos.

## Reflexión

La Programación Tradicional organiza el código en funciones separadas,
lo que funciona bien para problemas que solo son simples. La POO agrupa los datos
y las acciones dentro de una clase, haciendo el código más organizado
y fácil de escalar cuando el sistema crece o aumenta ayudándonos a poder encontrar
y resolver los problemas que se nos presenten de una manera más rápida y eficiente.
